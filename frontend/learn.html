Cryptographic Hashing

02

Block Structure

Each block contains: a list of transactions, a timestamp, the hash of the previous block, and its own hash. This forms an unbreakable chain.

Interactive Block

Each block's hash depends on the previous block. This is what makes the chain immutable.

03

Proof of Work Mining

Miners must find a special number (nonce) that makes the block's hash start with a certain number of zeros. This takes computational effort, securing the network.

04

Tamper Detection

If anyone tries to alter the data in a block, its hash changes, which breaks the link to the next block — making fraud immediately detectable.
